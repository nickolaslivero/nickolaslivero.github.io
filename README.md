# nickolaslivero.github.io

Portfólio pessoal hospedado no GitHub Pages com Tailwind CSS e conteúdo em português.

## Stack

- HTML estático
- Tailwind Play CDN (sem build step)

## Estrutura

- `index.html`: página única (PT/EN) com resumo profissional, habilidades, experiências, certificações e backlog para futuros projetos/TODOs. A troca de idioma é feita via toggle (PT/EN) e persistida em `localStorage`.

## Desenvolvimento

Edite `index.html` conforme necessário e faça o push. O GitHub Pages publicará automaticamente em `https://nickolaslivero.github.io`.
